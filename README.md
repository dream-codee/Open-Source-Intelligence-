# Open-Source-Intelligence-
End-to-end rumor detection pipeline using BERT, Bi-LSTM, and classical ML models on Twitter datasets.
## Features
- **Data Preprocessing:** Text cleaning, label encoding, context features (reply/retweet)
- **Classical ML:** SVM and Logistic Regression baselines
- **Deep Learning:** Bi-LSTM model for sequential text modeling
- **Transformers:** Fine-tuned BERT for multi-class rumor classification
- **Evaluation:** Classification reports, confusion matrices, t-SNE embedding visualization
- **Deployment Ready:** Saved fine-tuned BERT model and tokenizer
